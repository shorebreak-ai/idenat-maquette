**Retirer**

**Ajouter**
- **Confirmation immédiate** : récapitulatif + PDF du programme + délai de réponse + **interlocuteur nommé** (prénom + email). Relance J+3 si l'outil le permet.

**Garde-fou** : vérifier le **plugin de formulaire déjà actif** (CF7 / Gravity…) — **pas de nouveau plugin sans validation QG**.
